Movie Watchlist App

Project Description

This project is a web application that allows users to search for movies and view basic details such as the title, release year, and poster. It also includes a watchlist feature where users can save movies they want to watch later.

The main purpose of this project is to practice working with JavaScript, handling API data, and building an interactive user interface.

Features

* Search movies by name
* Display movie details such as title, year, and poster
* Add and remove movies from a watchlist
* Filter results based on type (movie or series)
* Sort movies by title or release year
* Responsive design for different screen sizes

API Used

OMDb API
http://www.omdbapi.com/

Technologies Used

* HTML
* CSS
* JavaScript (ES6)
* Fetch API

How to Run the Project

1. Clone or download the repository
2. Open the project folder
3. Open index.html in a web browser
4. Enter a movie name in the search bar to get results
